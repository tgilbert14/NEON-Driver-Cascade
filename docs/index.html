How it stays honest
Every app loads pre-bundled snapshots so there are no live-API waits, refreshes itself on a monthly off-peak schedule, and leads with transparent, well-sourced methods: detection-corrected abundance, honest diversity estimators, area-corrected detectability, interval-censored phenology, and data-quality flags you can click and download. The Cascade only lines up what each explorer already measures, on shared field sites, so the bottom-up story is built from real numbers, not a model stacked on a model.
