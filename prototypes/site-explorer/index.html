What drives this place

A few things really seem to run it

Each card is the plain-language read from the atlas' committed data. Tap “peel back the science” for the real numbers — because here, honesty is the interesting part.

This is a design prototype, not a published NEON product. It reads real per-site direction screens from the committed Driver Response Atlas bundle (data/cascade.rds), regenerated by export_data.py.
Read the numbers honestly. Single-site values are exploratory: with only ~6–13 years per site, no one site can be statistically significant — the direction is what's readable. The only result that clears significance is the pooled network test shown up top. The year-wheel is a schematic of each biome's typical rhythm, not measured monthly data.
