What drives this place?

NEON sends people to the same fixed patches of ground, year after year, to count the same things — plants, mice, beetles, birds — and record the weather over them. This is one of those places, taken apart down to the individual plant.

Built so far: Santa Rita, Arizona · one plot, 179 tagged plants · 45 more sites mapped, not yet built

Where these numbers came from. —
Read the numbers honestly. Single-site values are exploratory: with only ~6–13 years per site, no one site can be statistically significant — the direction is what's readable. The only result that clears significance is the pooled network test shown up top.
